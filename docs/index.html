infiniteScroll

Sets up infinite scrolling for WordPress

infiniteScroll(options: Options)
Parameters
options (Options) Options for setting up infinite scroll

Options

Options

Type: object

Properties
collection (string) : Backbone collection to use (defaults to Posts)
data (object) : Value of data object passed to fetch
container (string) : Container element selector
more (string) : More button element selector
classPrefix (string) : Class prefix used to deduplicate post elements
moreDisplay (string) : CSS display of the more button when it's visible
createElement (function) : Function that creates new elements given post attributes
